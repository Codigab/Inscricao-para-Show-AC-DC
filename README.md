## 📌 Sobre o Projeto

Este projeto consiste na criação de um **site para inscrição em shows da banda AC/DC**, utilizando tecnologias fundamentais do desenvolvimento web.

O objetivo da atividade foi aplicar conceitos práticos de:

- Estruturação de páginas com **HTML**
- Estilização com **CSS**
- Interatividade básica com **JavaScript**
- Uso de **elementos multimídia**
- Construção de **formulários completos**

---

## 🛠 Tecnologias Utilizadas

- ✅ HTML5  
- ✅ CSS3  
- ✅ JavaScript  
- ✅ Elementos multimídia (vídeo incorporado)  
- ✅ Campos de formulário  
- ✅ Radio buttons  
- ✅ Checkboxes  

---

## 🎯 Funcionalidades do Site

- 🎥 Vídeo incorporado do show
- 📝 Formulário de inscrição
- 🎫 Seleção de tipo de ingresso (Radio Button)
- 🎶 Seleção de preferências (Checkbox)
- 📅 Tabela com datas e horários dos shows
- 🎨 Estilização personalizada com tema dark

---

## 🧠 Conceitos Aplicados

- Separação de responsabilidades (HTML + CSS externo)
- Organização semântica da página
- Boas práticas em formulários
- Estruturação de tabelas
- Responsividade básica
